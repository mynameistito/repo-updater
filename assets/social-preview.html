repo-updater

Mass-update dependencies across all your git repos in one command. Auto-detects your package manager, commits changes, and opens PRs automatically.

⚡
Auto-detects npm, pnpm, yarn & Bun from lockfiles
🔀
Creates branches, commits & GitHub PRs automatically
👁
Dry-run mode — preview without executing anything
🌐
Cross-platform: Windows, macOS & Linux
